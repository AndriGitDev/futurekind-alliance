# The Commons Future Party

A fictional/speculative political party website prototype for **The Commons Future Party**, based on **Freedom Commons Federalism**.

> Maximum freedom. Shared abundance. No domination.

This is **not** a registered political organisation, active electoral campaign, donation vehicle, or voter-targeting operation. It deliberately contains no backend, auth, database, analytics, tracking, donation forms, or mailing-list capture.

## Stack

- Astro
- TypeScript
- Tailwind CSS v4 via `@tailwindcss/vite`
- Static output

## Pages

- `/` — home and quick explanation
- `/manifesto/` — public-facing manifesto narrative
- `/principles/` — scannable principles
- `/policies/` — ten policy pillars
- `/lab/` — naming, slogan, palette, and logo exploration
- `/about/` — fictional-project disclaimer and design constraints

## Development

```bash
npm install
npm run dev
```

## Verification

```bash
npm run check
npm run build
npm run preview
```

## Content direction

The visual direction is solarpunk optimism, civic seriousness, clean European public-service design, and science-fiction dignity. Avoid real political party marks, copyrighted science-fiction branding, manipulative campaign dark patterns, and authoritarian aesthetics.
